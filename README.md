# Hello World App

### In this project, I created a simple `Hello World App` using `ReactJs`.

Step 1: Create a New React App

````js
      npm create vite@latest
````


Step 2: Navigate to the App Directory

````js
      cd hello-world-app
````

Step 3: src/App.jsx

````js
      import React from 'react'

      function App() {
        return (
          <>
             <h1>Hello World</h1>
          </>
        )
      }
      
      export default App
````
